## কখনও ব্যবহার করেননি? এখান থেকে শুরু করুন।

যদি আপনি কখনও Obsidian-এর কথা না শুনে থাকেন বা Markdown নোট না লিখে থাকেন, চিন্তা নেই। এক অনুচ্ছেদে:

**Obsidian** একটি ফ্রি অ্যাপ ([obsidian.md](https://obsidian.md)) যা আপনার কম্পিউটারে `.md` ফাইলের (Markdown, সাধারণ টেক্সট) একটি ফোল্ডার পড়ে এবং একটি সুন্দর, আন্তঃসংযুক্ত নোটবুক হিসেবে প্রদর্শন করে। যেহেতু সবকিছু সাধারণ টেক্সট, **আপনার নোটগুলি চিরকাল আপনারই** — কোনও ক্লাউড লক-ইন নেই, কোনও পেইড সাবস্ক্রিপশন নেই, কোনও ভেন্ডর নেই যে অদৃশ্য হয়ে যেতে পারে। Mind++ Obsidian-এর কাঠামোকে ভিত্তি হিসেবে ব্যবহার করে কারণ এটি আজ বিদ্যমান ব্যক্তিগত জ্ঞানের সবচেয়ে টেকসই এবং গোপনীয় ফর্ম্যাট।

**Claude Cowork** হল Claude-এর ডেস্কটপ অ্যাপ ([claude.ai/download](https://claude.ai/download)) যা আপনার কম্পিউটারে ফাইল পড়তে ও লিখতে পারে। যখন আপনি Mind++ ইনস্টল করেন, Claude সেই ফোল্ডারে আপনার নোটগুলি সংগঠিত করার একটি নির্দিষ্ট উপায় শেখে — যাতে যখন আপনি বলেন "আমার রাহুলের সাথে একটি মিটিং হয়েছে", সে নোটটি লেখে, অ্যাকশন আইটেম বের করে এবং স্বয়ংক্রিয়ভাবে আপনার কাজের তালিকা আপডেট করে।

**আপনাকে ডেভেলপার হওয়ার দরকার নেই।** Git জানার দরকার নেই। YAML জানার দরকার নেই। Mind++ কাঠামো সামলায়; আপনি শুধু Claude-এর সাথে কথা বলেন।

## কেন Mind++?

- **কোনও ভেন্ডর লক-ইন নেই** — আপনার ডেটা সাধারণ `.md` ফাইল, যে কোনও জায়গায় যে কোনও এডিটরে পঠনযোগ্য
- **ডিফল্টভাবে গোপনীয়তা** — সবকিছু আপনার ফোল্ডারে, আপনার মেশিনে থাকে
- **Obsidian-নেটিভ** — গ্রাফ ভিউ, ব্যাকলিঙ্ক, সার্চ এবং ট্যাগ বাক্সের বাইরে কাজ করে
- **AI প্রসেস করে, মানুষ নিশ্চিত করে** — Claude প্রসেস করে; যে কোনও সরানো বা মোছার আগে আপনি নিশ্চিত করেন
- **সক্রিয়, প্রতিক্রিয়াশীল নয়** — অনুরোধ ছাড়াই মেয়াদোত্তীর্ণ কাজ এবং নীরব ক্লায়েন্টদের সামনে আনে
- **বহুভাষিক** — আপনি যে ভাষায় লেখেন প্লাগইন সেই ভাষায় কাজ করে; Claude আপনার সাথে মানিয়ে নেয়

## 5 মিনিটের দ্রুত শুরু

> 👉 **বিস্তারিত ধাপে ধাপে গাইড চান?** → [Wiki-তে ইনস্টলেশন গাইড](https://github.com/felipe-venturini/mind-plus-plus/wiki/Installation-Guide)

### 1. তিনটি টুল ইনস্টল করুন

| টুল | এটি কী করে | লিঙ্ক |
|-----|------------|------|
| **Claude Cowork** (প্রয়োজনীয়) | Claude অ্যাপ যা প্লাগইন চালায় | [claude.ai/download](https://claude.ai/download) |
| **Obsidian** (দৃঢ়ভাবে প্রস্তাবিত) | আপনার নোটের ভিজ্যুয়াল নেভিগেটর | [obsidian.md](https://obsidian.md) |
| **আপনার সিঙ্ক পছন্দ** (ঐচ্ছিক) | যে কোনও জায়গায় ব্যাকআপ এবং অ্যাক্সেস — একটি বেছে নিন: [Obsidian Git](https://github.com/Vinzent03/obsidian-git), Google Drive, iCloud Drive, OneDrive বা Dropbox | দেখুন [Sync Strategies](https://github.com/felipe-venturini/mind-plus-plus/wiki/Sync-Strategies) |

### 2. আপনার ভল্ট কোথায় থাকবে তা বেছে নিন

আপনার কম্পিউটারে একটি ফোল্ডার বেছে নিন। প্রতিটি নোট, মিটিং এবং সিদ্ধান্ত সেখানে সংরক্ষিত থাকবে।

- **সবচেয়ে সহজ** — `~/Documents/Mind++`-এর মতো একটি ফোল্ডার তৈরি করুন
- **ক্লাউডে স্বয়ংক্রিয় ব্যাকআপ** — ক্লাউড-সিঙ্ক করা অবস্থানের ভেতরে ফোল্ডার তৈরি করুন: `~/Google Drive/Mind++`, `~/OneDrive/Mind++`, `~/Dropbox/Mind++` বা `~/Library/Mobile Documents/com~apple~CloudDocs/Mind++` (iCloud)। Obsidian এবং Cowork একই ফাইল পড়বে, এবং ক্লাউড সার্ভিস সেগুলি সংস্করণ করবে এবং আপনার সমস্ত ডিভাইসে সিঙ্ক করবে।
- **পাওয়ার-ইউজার পথ** — আপনার ভল্টকে একটি ব্যক্তিগত GitHub রিপোতে সংস্করণ করতে [Obsidian Git প্লাগইন](https://github.com/Vinzent03/obsidian-git) ব্যবহার করুন (কমিট ইতিহাস, রোলব্যাক এবং Obsidian মোবাইল অ্যাপের মাধ্যমে Android/iOS সিঙ্ক দেয়)

> ট্রেড-অফ এবং প্রতিটি ইউজ কেসের জন্য সুপারিশের জন্য [Wiki-তে Sync Strategies পৃষ্ঠা](https://github.com/felipe-venturini/mind-plus-plus/wiki/Sync-Strategies) দেখুন।

### 3. Mind++ প্লাগইন ইনস্টল করুন

1. সর্বশেষ [`mind-plus-plus.plugin`](https://github.com/felipe-venturini/mind-plus-plus/releases/latest) ফাইল ডাউনলোড করুন
2. Claude Cowork খুলুন → **Plugins**
3. `.plugin` ফাইলটি প্লাগইন এলাকায় টেনে আনুন (বা ইনস্টল ক্লিক করুন)

### 4. Cowork-এ আপনার ভল্ট ফোল্ডার নির্বাচন করুন

Claude Cowork-এ, ধাপ 2-এ বেছে নেওয়া ফোল্ডারটিকে আপনার ওয়ার্কস্পেস হিসেবে নির্বাচন করুন। Claude সরাসরি সেখানে পড়বে এবং লিখবে।

### 5. সেটআপ চালান

Claude-কে বলুন:

> **"আমার দ্বিতীয় মস্তিষ্ক সেটআপ করো"**

সেটআপ আপনার কাজের প্রসঙ্গ সম্পর্কে জিজ্ঞাসা করবে (ক্লায়েন্ট, পণ্য, টুল), ফোল্ডার কাঠামো তৈরি করবে, একটি ব্যক্তিগতকৃত `CLAUDE.md` লিখবে এবং দৈনিক ব্রিফ ও সাপ্তাহিক পর্যালোচনা নির্ধারণ করার প্রস্তাব দেবে।

### 6. ব্যবহার শুরু করুন

- প্রতি সকালে **"brief"** বলুন → Claude আজকের এজেন্ডা, মেয়াদোত্তীর্ণ কাজ এবং ফলো-আপ দেখাবে
- প্রতিটি কলের পরে **"আমি X-এর সাথে একটি মিটিং করেছি"** বলুন → Claude নোটটি লিখবে এবং আপনার কাজ আপডেট করবে
- প্রতি শুক্রবার **"weekly review"** বলুন → Claude সপ্তাহটি একত্রিত করে

ব্যস। আপনি যত বেশি ব্যবহার করবেন, আপনার ভল্ট তত বেশি একটি প্রকৃত দ্বিতীয় মস্তিষ্কে পরিণত হবে।

### 2টি এজেন্ট (agents)

এজেন্ট হল স্বায়ত্তশাসিত সাব-প্রসেস যা বিচ্ছিন্ন প্রসঙ্গে চলে এবং একটি পরিষ্কার ফলাফল ফেরত দেয়।

| এজেন্ট | এটি কী করে | কখন চলে |
|--------|------------|---------|
| `vault-researcher` | মূল কথোপকথনকে ওভারলোড না করে অনেক ফাইল জুড়ে গভীর পঠন | স্বয়ংক্রিয়ভাবে, যখন `knowledge-search` >15 ফাইল মেলে |
| `vault-auditor` | সম্পূর্ণ ভল্ট স্বাস্থ্য পরীক্ষা: মেয়াদোত্তীর্ণ কাজ, নীরব ক্লায়েন্ট, অনাথ ফাইল, পুরানো সিদ্ধান্ত | অনুরোধে ("আমার ভল্ট অডিট করো") বা সাপ্তাহিকভাবে নির্ধারিত |

| কাজ | ডিফল্ট সময়সূচী | স্কিল |
|-----|----------------|-------|
| দৈনিক ব্রিফ | সকাল 8:00, কর্মদিবস | `daily-brief` |
| সাপ্তাহিক পর্যালোচনা | বিকাল 5:00, শুক্রবার | `weekly-review` |
| ইমেল পাইপলাইন | প্রতি 2 ঘন্টায় | `process-meeting-emails` |
| ভল্ট অডিট | সকাল 9:00, সোমবার | এজেন্ট `vault-auditor` |

আপনি পরে সময়সূচী তৈরি, সম্পাদনা বা মুছতে পারেন — শুধু Claude-কে বলুন।

## আপনার ভল্ট কীভাবে সংগঠিত হয়

Mind++ এই কাঠামো তৈরি করে এবং বজায় রাখে। আপনাকে এটি মুখস্থ করতে হবে না — Claude জানে কোথায় কী যায়।

```
clients/{client}/       ← প্রতি ক্লায়েন্টে একটি ফোল্ডার
  {Client}.md           ← পরিচিতি, সক্রিয় প্রকল্প, স্ট্যাটাস সহ ড্যাশবোর্ড
  meetings/             ← YYYY-MM-DD শিরোনাম.md
  decisions/            ← ADRs — NNN শিরোনাম.md
  references/           ← ব্রিফিং, স্প্রেডশিট, সাপোর্ট ডক
  Tasks {Client}.md     ← খোলা কাজের ট্র্যাকার

products/{product}/     ← আপনার নিজস্ব পণ্য (যদি থাকে)
operational/            ← অভ্যন্তরীণ কাজ, দল, কনফিগ
personal/               ← ব্যক্তিগত প্রকল্প, লক্ষ্য, নোট
shared/                 ← টেমপ্লেট, প্রম্পট, সংযুক্তি
inbox/                  ← অনিশ্চিত আইটেম — process-inbox দিয়ে ট্রায়েজ
archive/                ← বন্ধ আইটেম — এখানে সরানো, কখনও মুছে ফেলা হয় না
```

আরও বিবরণ [Wiki](https://github.com/felipe-venturini/mind-plus-plus/wiki)-তে।

## গোপনীয়তা

সবকিছু আপনার কম্পিউটারে থাকে। Mind++ কোথাও আপনার ডেটা পাঠায় না। একমাত্র বাহ্যিক পরিষেবাগুলি জড়িত যা আপনি স্পষ্টভাবে Cowork-এ সংযুক্ত করেন (উদাহরণস্বরূপ, মিটিং পাইপলাইনের জন্য Gmail, দৈনিক ব্রিফের জন্য Google Calendar) — এবং সেগুলি অপ্ট-ইন।

যদি আপনি Obsidian Git বা একটি ক্লাউড-সিঙ্ক করা ফোল্ডার (Drive, OneDrive, Dropbox, iCloud) ব্যবহার করেন, আপনার নোটগুলি সেই পরিষেবাতে প্রতিলিপি হয় — কিন্তু আপনি এটি বেছে নিয়েছেন এবং আপনি এটি নিয়ন্ত্রণ করেন।

আপনার ইউজার প্রোফাইল এবং স্মৃতি ভল্টের ভেতরে একটি লুকানো `.auto-memory/` ফোল্ডারে স্থানীয়ভাবে বাস করে।

## প্রায়শই জিজ্ঞাসিত প্রশ্ন

সংক্ষিপ্ত তালিকা — সম্পূর্ণ FAQ [Wiki](https://github.com/felipe-venturini/mind-plus-plus/wiki/FAQ)-তে।

**আমাকে কি Markdown জানতে হবে?** না। Claude আপনার জন্য Markdown লিখবে। আপনি স্বাভাবিক ভাষায় কথা বলেন।

**আমি কি Obsidian ছাড়া ব্যবহার করতে পারি?** হ্যাঁ — প্লাগইনের শুধু ফোল্ডার দরকার। Obsidian দৃঢ়ভাবে প্রস্তাবিত কারণ গ্রাফ ভিউ এবং ব্যাকলিঙ্ক ভল্টকে নেভিগেবল করে, কিন্তু প্রযুক্তিগতভাবে আপনি যে কোনও এডিটরে ফাইল খুলতে পারেন।

**আমি কি ইংরেজি ছাড়া অন্য ভাষায় ব্যবহার করতে পারি?** হ্যাঁ। সেটআপের সময় (বা যে কোনও সময়) Claude-কে আপনার পছন্দের ভাষা বলুন, এবং সে সেই ভাষায় নোট, ব্রিফ এবং পর্যালোচনা লিখবে। প্লাগইনের ফোল্ডার কাঠামো ডিজাইন অনুসারে ইংরেজি নাম ব্যবহার করে (পোর্টেবিলিটির জন্য), কিন্তু আপনার বিষয়বস্তু আপনার ভাষায়।

**যদি আমার ইতিমধ্যে Obsidian-এ নোট থাকে?** Mind++-কে আপনার বিদ্যমান ভল্টে নির্দেশ করুন। সেটআপ বিদ্যমান ফাইলগুলি সনাক্ত করে এবং একত্রিত করে, ওভাররাইট করে না।

**আমি কীভাবে মোবাইলে আমার নোট অ্যাক্সেস করব?** Obsidian Git ব্যবহার করুন (একটি ব্যক্তিগত GitHub রিপোতে সিঙ্ক করে যা আপনি Obsidian মোবাইল অ্যাপের মাধ্যমে অ্যাক্সেস করেন), অথবা আপনার ভল্টকে একটি ক্লাউড ফোল্ডারে (Drive / OneDrive / Dropbox / iCloud) রাখুন এবং আপনার ফোনে খুলুন।
